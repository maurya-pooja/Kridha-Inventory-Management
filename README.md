**Kridha Inventory & Billing System** 
Kridha Inventory and Billing System is a professional full-stack web application designed to help small businesses manage their stock sales efficiently. it is built using React, Node.js, and MySQL.


## Technologies Used
* Frontend: React.js, Material UI, CSS
* Backend : Node.js, Express.js
* Database: MYSQL
* Other Tools: Axios, React Router

## Key Features
* Inventory Control: Easily add, update, and delete products from the system.
* Real-time Stock Tracking: stock levels update automatically after every sale.
* Billing: Generate instant bills for customer with autometic total calculations.
* Sales History : A dedicated section to view and manage all past transaction records.
* Secure Login: Features an authentication system to keep shop data safe.
* Responsive Design: The application is fully optimized for both mobile and desktop screens.


## Project Structure
* The project is organized into two main folders:
*/InventoryControlSystem: This Contains the complete Frontend (React) code.
/Inventory-backend: Contains the Backend (Node.js) and Database configiuration.

## Purpose 
* The goal of this project is to solve a real-world problem using core front-end back-end with database technologies.
